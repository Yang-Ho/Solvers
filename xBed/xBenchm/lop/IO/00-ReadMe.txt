Files in this directory have been downloaded from
http://www.optsicom.es/lolib/

For compatibility with xBed conventions, the name of each file 
is slightly modified without loosing the original context of the name.

For example, under the directory lop/IO/:

N-t70l11xx  —> N-44-t70l11xxN-t70x11xx  —> N-44-t70x11xxN-t75n11xx  —> N-44-t75n11xxN-t75u11xx  —> N-44-t75u11xxN-be75eec   —> N-50-be75eecN-be75np    —> N-50-be75npN-be75oi    —> N-50-be75oiN-tiw56r54  —> N-56-tiw56r54N-usa79     —> N-79-usa79
etc.

Under xBed convention, the second substring that follows '-' is interpreted 
as the original dimension of the instance coordinate — before any reductions 
due to instance symmetries, etc.

Thus:

N-44-t70l11xx  implies a permutation coordinate of length  44
N-44-t70x11xx  implies a permutation coordinate of length  44
N-50-be75eec   implies a permutation coordinate of length  50
N-79-usa79     implies a permutation coordinate of length  79
etc.
